# Credits API

Check your credit balance via API.

## GET /v1/credits

Returns the current credit balance for the authenticated user.

**Request:**

```bash
curl https://keyaos.com/v1/credits \
  -H "Authorization: Bearer YOUR_API_KEY"
```

**Response:**

```json
{
  "data": {
    "balance": 42.50
  }
}
```

## Response Fields

| Field | Type | Description |
|---|---|---|
| `balance` | `number` | Remaining credits in USD-equivalent |
